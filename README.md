6-كل فقره من فقرات الإحتبار محدده بوقت زمني وهي 3 دقائق لكل فقره حسب المعايير العالمية المتبعه لهذه النوعية من الإختبارات.

7-قابل للتحديث.

8-محاكاة تفاعلية لمسائل خاصة أكثر من رائع.
